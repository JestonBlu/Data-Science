Welcome to my Data Science site

This website is intended to be a personal reference for data science related tasks. It consists of reproducible programming examples for various analytic and visualization techniques. In the sidebar menu, an icon next to each item identifies the source language used to create each example.

I am not a web programmer and I pretty much use Google Chrome exclusively for testing so I recommend it for viewing this website especially for the javascript visualizations.

All code is available on my Github project page.

Computer Languages and Tools

All R examples have been generated using the Rmarkdown Package which generates HTML reports directly from R code and also allows for markdown style formatting. It is a very handy package for generating reproducible reports.
My primary interest in javascript is for interactive visualization. One particular visualization library of interest is D3.
Julia is a new developing language for technical computing. It is fast, flexible, and shows a lot of promise. I have experimented with it off and on, but so far I have no reproducible examples.
RStudio is the best IDE for R in my opinion. It has a nice integration with rmarkdown.
Atom is my text editor of choice because of its nice integration with github.
